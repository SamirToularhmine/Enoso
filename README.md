#Requirements
###Pour la GUI (python)
- Python3 
- Tkinter (installer par défaut avec python)
- Pillow (pip3 install Pillow) 
- graphviz (commande ```$ dot -Tpng``` nécessaire)

Vérifier l'url du JAR dans la variable JAR_LOCATION en haut de gui.py.

Pour start via GUI :  
```$ python gui.py```
